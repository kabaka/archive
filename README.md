## Development

### Web UI

Clone the repository:

```sh
git clone https://github.com/kabaka/archive.git
cd archive
```

Build the code:

```sh
cd packages/archive-types
yarn build
cd ../archive-core
yarn build
```

Start the application and watch for changes to rebuild:

```sh
cd packages/archive-types
yarn dev
cd ../archive-web
yarn dev
```
